# Listify

Prototype aplikasi untuk mencatat barang yang ingin dibeli, sedang diproses, atau sudah dibeli.

## Fitur
- Multi notes
- Add / delete notes
- Add / update / delete item
- Search
- Filter status
- Total harga
- Progress bar
- Dark / light mode
- Export CSV / JSON
- Import JSON
- localStorage persistence

## Run
```bash
npm install
npm run dev
